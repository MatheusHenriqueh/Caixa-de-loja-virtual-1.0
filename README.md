## Descrição

Este é um **projeto simples em Python** que simula uma **calculadora de preços para uma loja**.  
O usuário informa os preços de até três produtos, e o programa calcula o **total da compra**, exibindo o valor final de forma clara e formatada.

O objetivo é praticar conceitos básicos de Python, como:  
- Variáveis (`float`, `str`)  
- Entrada de dados (`input`)  
- Operações matemáticas (`+`)  
- Estruturas de saída (`print` e f-strings)  

---

## Funcionalidades

- Inserir preços de produtos via terminal  
- Soma automática dos produtos  
- Exibição do total com duas casas decimais  
- Mensagens amigáveis para o usuário
